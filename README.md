# Página de la UTSJR Mejorada

Este es un conjunto de diferentes subpáginas de la UTSJR reimaginadas usando tecnologías de desarrollo web
modernas (por ejemplo, React, TypeScript y Tailwind).

Originalmente, el proyecto fue concebido con la idea de rediseñar la página de inicio del portal de alumnos de la UTSJR, 
pero después de meditar un tiempo, se ha decidido expandir su alcance para abarcar más subpáginas y tener una idea más
clara de cómo sería la presencia digital de la universidad si invirtiera más en su página web.
